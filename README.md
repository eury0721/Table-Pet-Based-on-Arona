所有资源和设置均在Resource，初次运行前，必需先到Resource\AronaDSText.txt进行AI对话的初始设置deepseekPhoneNumber和deepseekPassWord，是deepseek的账号和密码（不是api），AI对话依赖在网页版的deepseek上。

配音依赖本地的GPT-soVITS。

并下载GPT-soVITS运行的模型和参考音频。要求与api.py在同一目录下。https://pan.baidu.com/s/1x1Ur9qbarrikXPBw8IVsBg?pwd=1234 提取码: 1234
